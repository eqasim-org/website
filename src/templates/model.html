The eqasim model

The base configuration of eqasim features a pipeline that applies common algorithms to synthesize rich and detailed agent-based transport simulation scenarios.

The scenarios are simulated using the agent- and activity-based transport simulation framework MATSim in combination with eqasim's novel Discrete Mode Choice extension.

All parts in that chain are replacable and adjustable which allows to test novel, specific models in an integrated context.
